## Biimplication

```agda
⇔-equiv₁
  : ∀ {Γ} {φ ψ}
  → Γ ⊢ φ ⇔ ψ
  → Γ ⊢ (φ ⇒ ψ) ∧ (ψ ⇒ φ)

⇔-equiv₂
  : ∀ {Γ} {φ ψ}
  → Γ ⊢ (φ ⇒ ψ) ∧ (ψ ⇒ φ)
  → Γ ⊢ φ ⇔ ψ

  : ∀ {Γ} {φ ψ}

    : ∀ {Γ} {φ ψ ω}

⇔-assoc
```

## Disjunction.

```agda
∨-assoc₁

  → Γ ⊢ φ ∨ (ψ ∨ ω)
  → Γ ⊢ (φ ∨ ψ) ∨ ω

∨-assoc₂
  : ∀ {Γ} {φ ψ ω}

∨-comm
  : ∀ {Γ} {φ ψ}
  → Γ ⊢ φ ∨ ψ
  → Γ ⊢ ψ ∨ φ

∨-dist₁
  : ∀ {Γ} {φ ψ ω}
  → Γ ⊢ φ ∨ (ψ ∧ ω)
  → Γ ⊢ (φ ∨ ψ) ∧ (φ ∨ ω)

∨-dist₂
  : ∀ {Γ} {φ ψ ω}
  → Γ ⊢ (φ ∨ ψ) ∧ (φ ∨ ω)
  → Γ ⊢ φ ∨ (ψ ∧ ω)

∨-dist
  : ∀ {Γ} {φ ψ ω}
  → Γ ⊢ (φ ∨ (ψ ∧ ω)) ⇔ ((φ ∨ ψ) ∧ (φ ∨ ω))

∨-equiv
  : ∀ {Γ} {φ ψ}
  → Γ ⊢ φ ∨ ψ
  → Γ ⊢ ¬ (¬ φ ∧ ¬ ψ)
∨-to-¬¬∧¬ = ∨-equiv

∨-dmorgan₁

  → Γ ⊢ ¬ (φ ∨ ψ)

∨-dmorgan₂

  → Γ ⊢ ¬ φ ∧ ¬ ψ

∨-dmorgan

  → Γ ⊢ ψ ∨ φ → Γ ⊢ ¬ φ ∨ ω
  → Γ ⊢ ψ ∨ ω

  : ∀ {Γ} {φ ψ ω}
  → Γ ⊢ φ ∨ ψ → Γ ⊢ ω ∨ ¬ φ
  → Γ ⊢ ψ ∨ ω

resolve₃
  : ∀ {Γ} {φ ψ ω}
  → Γ ⊢ ψ ∨ φ → Γ ⊢ ω ∨ ¬ φ

resolve₅

  → Γ ⊢ ψ

resolve₇
```

## Negation

```agda
¬-equiv₁
  : ∀ {Γ} {φ}
  → Γ ⊢ ¬ φ
  → Γ ⊢ φ ⇒ ⊥
¬-to-⇒⊥ = ¬-equiv₁

¬-equiv₂
  : ∀ {Γ} {φ}
  → Γ ⊢ φ ⇒ ⊥
  → Γ ⊢ ¬ φ
⇒⊥-to-¬ = ¬-equiv₂

¬¬-equiv₁
  : ∀ {Γ} {φ}
  → Γ ⊢ ¬ (¬ φ)
  → Γ ⊢ φ

¬¬-equiv₂
  : ∀ {Γ} {φ}
  → Γ ⊢ φ
  → Γ ⊢ ¬ (¬ φ)

¬∨-to-⇒
  : ∀ {Γ} {φ ψ}
  → Γ ⊢ ¬ φ ∨ ψ
  → Γ ⊢ φ ⇒ ψ

¬⊤-to-⊥
  : ∀ {Γ}
  → Γ ⊢ ¬ ⊤
  → Γ ⊢ ⊥

⊤-to-¬⊥
  : ∀ {Γ}
  → Γ ⊢ ⊤
  → Γ ⊢ ¬ ⊥

¬⊥-to-⊤
  : ∀ {Γ}
  → Γ ⊢ ¬ ⊥
  → Γ ⊢ ⊤

⊥-to-¬⊤
  : ∀ {Γ}
  → Γ ⊢ ⊥
  → Γ ⊢ ¬ ⊤

¬-inside
  : ∀ {Γ} {φ ψ}
  → φ ≡ ψ
  → Γ ⊢ ¬ φ
  → Γ ⊢ ¬ ψ
≡-¬-to-¬ = ¬-inside

subst⊢¬
  : ∀ {Γ} {φ ω}
  → Γ ⊢ ω ⇒ φ
  → Γ ⊢ ¬ φ
  → Γ ⊢ ¬ ω

```
